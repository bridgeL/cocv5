# 任务计划：修复总价格查询 Bug

## 任务列表

### 1. 复现 Bug
- 启动后端服务
- 使用前端或测试脚本发送消息"告诉我苹果的总价格后"
- 记录完整的错误堆栈信息

### 2. 分析 Bug 原因
- 根据错误堆栈定位问题代码
- 分析问题产生的根本原因

### 3. 修复 Bug
- 修改对应代码文件
- 确保修复不会引入新的问题

### 4. 自测
- 重新测试相同的消息
- 验证工具调用和结果计算正常
- 测试其他相关场景确保没有回归

### 5. 提交代码
- 按 git 规范提交修复

## 变更文件
- 根据 Bug 原因确定（可能是 agent.py、memory.py 或 app.py）
