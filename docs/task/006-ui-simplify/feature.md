# 需求文档：UI 简化

## 需求背景

当前前端界面采用渐变色背景、紫色主题、彩色头像等花哨设计，视觉元素过于复杂，不符合简洁高效的工具型产品定位。

## 需求目标

将界面改为简洁清爽的设计风格，减少视觉干扰，突出内容本身。

## 具体需求

### 1. 背景
- 移除渐变色背景
- 改为纯白色背景

### 2. 消息气泡
- 用户消息：浅色纯色气泡（如浅灰色），黑色文字
- AI 消息：白色或极浅色气泡，黑色文字
- 移除彩色头像、渐变效果

### 3. 工具调用气泡
- 改为简洁的边框或浅色背景
- 黑色文字
- 移除彩色装饰元素

### 4. 整体风格
- 以黑白灰为主色调
- 保持清晰的信息层级
- 保持现有的布局结构（顶部状态栏、聊天区、输入框等）

## 预期效果

界面清爽简洁，无多余视觉元素，用户注意力集中在对话内容上。
