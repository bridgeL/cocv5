# 功能需求：AI 单轮对话

## 功能描述

实现前端通过 WebSocket 与 AI 进行单轮对话的功能。前端发送消息，后端转发给 AI 模型，并将 AI 的流式响应实时转发回前端。

## 业务场景

用户在前端页面输入消息（如 "hello"），后端将消息发送给 DeepSeek AI 模型，AI 以流式方式返回响应，后端将每个 chunk 即时推送给前端展示。

## 验收标准

1. **消息转发**
   - 前端发送类型为 `chat` 的消息
   - 后端接收后将内容转发给 DeepSeek AI

2. **流式响应**
   - 后端使用流式 API 调用 AI
   - 每收到一个 chunk 立即转发给前端
   - 前端实时显示 AI 回复内容

3. **消息格式**
   - 请求：`{ "type": "chat", "message": "hello" }`
   - 响应：`{ "type": "chat_chunk", "content": "..." }`
   - 结束标记：`{ "type": "chat_done" }`

4. **配置支持**
   - 支持通过环境变量配置 DeepSeek API 密钥和模型

## 非功能性需求

- 响应延迟：首包响应时间 < 2 秒
- 支持标准的 OpenAI 兼容接口格式
