# Agent 接口文档

本文档描述 AI Agent 系统的 HTTP 接口。

---

## 接口列表

### 1. 首页

返回前端页面 `index.html`。

#### 接口信息

- **接口名**: 首页
- **接口方法**: GET
- **接口地址**: `/`

#### 请求参数

无

#### 正常返回

```
HTTP 200 OK
Content-Type: text/html

<html>...</html>
```

#### 异常返回

| code | msg | 说明 |
|------|-----|------|
| 1111 | 文件不存在 | index.html 文件缺失 |

---

## 接口规范说明

### 返回值格式

所有 JSON 接口返回值必须是字典，包含以下字段：

- `code`: 四位数字的字符串
  - `0000` 代表默认正常返回
  - `1111` 代表默认失败返回
  - `0xxx` 开头的是正常返回
  - `1xxx` 开头的是异常返回
- `msg`: 失败原因描述
- `data`: 返回的具体数据

### 请求方式规范

- POST 传递参数时必须是 body JSON
- GET 传递参数时必须是 query param
